## Critical historical bug that MUST NOT return

There was a production-breaking bug caused by **double JSON serialization**.

Broken flow:

* frontend sent `JSON.stringify({ key, value })`
* API did `kvSet(key, JSON.stringify(value))`
* kvSet also stringified again

This caused KV values like:
`'"[{...}]"'`

instead of:
`[{...}]`

This broke production because GET returned:

```json
"value": "[{...}]"
```

instead of:

```json
"value": [{...}]
```

This then caused frontend failures like:

* `Array.isArray(value) === false`
* `.filter is not a function`
* mobile and incognito showing wrong or empty data
* admin appearing correct only because localStorage masked the bug
