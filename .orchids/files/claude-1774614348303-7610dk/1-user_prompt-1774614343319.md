Additional requirement — Notification Bell + Sound for New Leads

Please also add a proper notification system for new incoming leads.

### Requirements:

1. Notification Bell (Admin Header)

* Add a visible bell icon in the admin top bar
* It must always be clearly visible (not hidden / low contrast)

2. Unread Leads Indicator

* Show a badge (number or dot) on the bell when new leads arrive
* The badge should represent unread/new leads

3. Real-Time Update
   When a new lead is received:

* the bell must update immediately
* the unread indicator must appear or increase

4. Sound Notification

* A short notification sound must play when a new lead arrives
* Must work while the admin panel is open
* Should not be aggressive — just a clear, short chime

5. Interaction

* Clicking the bell should:

  * open the leads view or a dropdown
  * mark leads as “viewed” (optional but recommended)

6. Persistence

* Unread state should not reset randomly on refresh
* Only reset when leads are actually viewed

This must work together with the existing toast notifications and not replace them.
