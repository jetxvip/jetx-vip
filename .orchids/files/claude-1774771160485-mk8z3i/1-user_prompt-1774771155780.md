JetX.vp — Redesign the “Sold Out” Visual State (More Dramatic / Premium)

The current Sold Out implementation is NOT the desired design.

Right now it only shows a small red label/badge in the top corner, which is too weak and not visually impactful enough.

I want the Sold Out state to feel dramatic, obvious, and emotionally clear — as if the visitor understands immediately that this opportunity has already been taken.

## Required redesign

For both:

* Empty Leg flights
* Special Offers

When an item is marked as Sold Out, do NOT use only a small badge in the corner.

Instead, apply a stronger visual treatment.

### Required visual direction

I want:

* a **clear red diagonal line** (or strong red strike treatment) across the card
* and on top of that line, large **SOLD OUT** text in **capital letters**
* the treatment should be impossible to miss

The goal is for the user to instantly feel:

* this was available
* this is no longer available
* someone else already took it

## Design requirements

The Sold Out treatment must be:

* dramatic
* elegant
* premium
* highly visible
* still clean and not cheap-looking

It should NOT look like a generic ecommerce sticker.

### Suggested behavior

* one diagonal red band across the card
  or
* one strong horizontal/diagonal strike line through the main offer area

And on that band/line:

* **SOLD OUT** in large uppercase letters
* high contrast
* centered or prominently placed

## Important

The Sold Out state must dominate the card visually.

The current small top badge is not enough.

Visitors should immediately recognize the card as closed/unavailable before they even try to interact with it.

## CTA behavior

* booking/request CTA must not behave like an active item
* either disable it or replace it with a non-booking state
* but the main visual emphasis should be the dramatic Sold Out treatment, not just button changes

## Extra direction

You may slightly mute or fade the card underneath, but:

* the content must still remain visible
* the offer should still look real and premium
* the Sold Out treatment should sit above it clearly

## Final requirement

Please redesign the Sold Out state so it feels like a missed opportunity / closed deal, not just a small informational label.
