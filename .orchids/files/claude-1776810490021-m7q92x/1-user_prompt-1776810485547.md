# SEO Pages Cleanup + Safe Linking Strategy

We are simplifying the SEO pages to eliminate broken links and improve quality.

## Part 1 — Clean SEO Pages

Update `SeoPageTemplate.jsx` and related logic.

### Changes required:

1. REMOVE:

   * relatedSlugs section completely
   * any internal SEO-to-SEO links
   * any buttons that point to dynamic slugs

2. KEEP:

   * ONE primary CTA per page only

3. CTA behavior:

   * must point to an existing safe route:
     `/quote` OR `/#quote`
   * must NEVER point to `/[lang]/[slug]`

4. Result:

   * ZERO broken links
   * ZERO dynamic internal links
   * clean, premium landing pages
