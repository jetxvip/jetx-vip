# Production Bug — YouTube Audio Blocked by CSP + Additional Issues

There are multiple production issues.

---

## 1. ROOT CAUSE FOUND — YouTube is BLOCKED by CSP

Production console error:

Refused to load the script 'https://www.youtube.com/iframe_api'
because it violates Content Security Policy

This means:

* The YouTube player cannot load at all
* Therefore audio never starts
* This is NOT a playback bug — it is a CSP restriction

### Required fix:

Update Content Security Policy to allow YouTube.

Must allow:

script-src:
https://www.youtube.com
https://www.youtube-nocookie.com

frame-src:
https://www.youtube.com
https://www.youtube-nocookie.com

media-src:
https://www.youtube.com
https://*.googlevideo.com

connect-src:
https://www.youtube.com
https://*.googlevideo.com

Do NOT remove existing security.
Only extend CSP safely.

Check:

* next.config.js headers()
* middleware.ts
* vercel.json
