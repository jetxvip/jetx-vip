JetX.vp — Fleet Page Hover Interaction + Visual Enhancement

I want to improve the aircraft icons/cards in the Fleet section, both on:

* the dedicated Fleet page
* and the Fleet section on the homepage

## 1. Hover animation for aircraft items

Currently, the aircraft items/icons are static.

I want them to feel more interactive so users immediately understand that they can click them to view more information.

### Requirement

When hovering over an aircraft item with the mouse:

* it should animate subtly
* it should visually “lift” or “pop out” slightly
* it should feel like an interactive button/card
* users should clearly understand that the item is clickable

### Design direction

The effect should be:

* elegant
* subtle
* premium
* smooth
* not exaggerated or playful

### Suggested hover behavior

On hover:

* slight scale up
* slight lift / translateY
* soft shadow increase
* optional subtle glow / emphasis
* optional gentle image movement

Important:

* keep it refined
* no cheap or aggressive animation
* no jitter
* no heavy motion

This should work especially on desktop hover states.
