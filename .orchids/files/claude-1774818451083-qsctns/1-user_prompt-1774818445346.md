JetX.vp — Enforce client creation before closing a lead + quick action button

We need to improve the admin workflow to ensure data integrity and proper CRM behavior.

## Problem

Currently, an admin can close a lead (Confirmed / Confirmed Closed) even if the person is NOT yet a client.

This creates major issues:

* flight history is not properly linked
* revenue tracking becomes inaccurate
* client records are incomplete or missing

This must be prevented.

---

## Required behavior

Before allowing a lead to be finalized (closed/confirmed), the system MUST check:

Is this person already a client?

---

## If the person is NOT a client

Block the closing action and show a clear red warning:

**This person is not a client yet. Please make him a client first before closing.**

---

## NEW: Inline quick action (critical UX improvement)

Inside the same warning / modal / area, add a button:

**[ Create Client Now ]**

### Behavior of the button:

When clicked:

* automatically create a new client using the lead's existing data:

  * name
  * email
  * phone
* add the client to the clients system immediately
* link the lead to that client

After creation:

* allow the admin to continue the closing process WITHOUT restarting the flow

---

## Important logic rules

* If client already exists → allow closing normally
* If not → block closing until client is created
* Client creation must be:

  * instant
  * reliable
  * persisted (KV + sync like other data)

---

## UX expectations

* No page reload required
* Smooth inline flow
* No data loss
* Admin stays in the same screen

---

## Do NOT break

Do not affect:

* existing lead system
* KV sync
* notifications
* admin auth
* existing client structure

---

## Expected result

1. Admin tries to close a lead
2. System checks client existence
3. If not exists:

   * show red warning
   * show "Create Client Now" button
4. Admin clicks button:

   * client is created instantly
   * lead is now linked
5. Admin continues closing process successfully

This must feel fast, intuitive, and reliable.
